# Factorial Program in Python

This is a beginner-level Python program that calculates the factorial of a given number using a loop.

## 📌 Description
- The program takes an integer input from the user.
- If the number is negative, it displays an appropriate message.
- If the number is zero or positive, it calculates the factorial using a for loop.
- Finally, it prints the result.

## 💻 How It Works
Factorial of a number (n) is calculated as:

n! = n × (n-1) × (n-2) × ... × 1

Example:
If the input is 5  
Output will be: 120

## 🛠 Concepts Used
- User Input
- Conditional Statements (if-else)
- For Loop
- Basic Arithmetic Operations
